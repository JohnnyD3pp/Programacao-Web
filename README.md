# Programação Web

Este repositório reúne exercícios, exemplos e atividades desenvolvidos na disciplina de **Programação Web**, com foco em HTML, CSS, Bootstrap, manipulação do DOM e consumo de APIs com JavaScript.

O material registra a evolução dos estudos em desenvolvimento front-end, desde a construção de páginas estáticas até exemplos com formulários, layout responsivo e requisições com `fetch`.

## Conteúdos Abordados

- Estrutura básica de documentos HTML
- Estilização com CSS
- Layouts com colunas, fundos e imagens
- Componentes e formulários com Bootstrap
- Manipulação de elementos do DOM
- Eventos em JavaScript
- Consumo de APIs com `fetch`
- Criação e listagem dinâmica de dados em HTML
- Tratamento básico de conteúdo com encode

## Descrição das Pastas

### DOM

Contém atividades voltadas à criação de formulários, entrada de dados e interação com elementos da página.

Principais temas:

- Campos de formulário
- Inputs de texto, e-mail e número
- Organização visual com Bootstrap
- Manipulação de dados digitados pelo usuário

### Exemplo do Professor

Reúne arquivos de referência utilizados em aula, com exemplos de HTML e CSS.

Principais temas:

- Limites e fundos
- Colunas com fundo pintado
- Estruturação visual de páginas
- Práticas iniciais com HTML e CSS

### Exercício Fetch

Contém exercícios práticos com uso de `fetch`, simulando criação e listagem de dados relacionados a jogos, livros e posts.

Principais temas:

- Formulários para cadastro
- Envio de dados para API
- Listagem de registros
- Uso de `encodeURIComponent`
- Integração entre HTML, JavaScript e API

### Exercícios

Reúne páginas HTML desenvolvidas como prática de estruturação e estilização.

Exemplos de atividades:

- Página de currículo
- Página de receita
- Páginas temáticas sobre filmes, jogos e veículos
- Uso de imagens locais
- Estilização com CSS interno

### Fetch

Contém exemplos introdutórios de requisições HTTP com JavaScript.

Principais temas:

- Listagem simples com `fetch`
- Geração de HTML dinâmico
- Uso de funções assíncronas
- Exibição de dados retornados por API
- Tratamento de caracteres especiais com encode

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap**
- **Fetch API**
- **DOM**

## Como Visualizar os Arquivos

Os arquivos do repositório são páginas HTML. Para visualizar qualquer atividade:

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `.html` desejado.
4. O arquivo será aberto no navegador.

Também é possível abrir os arquivos pelo Visual Studio Code usando a extensão **Live Server**.

## Observação

Alguns exemplos utilizam requisições para APIs externas. Nesses casos, é necessário estar conectado à internet para que a listagem ou o envio de dados funcione corretamente.

Este repositório tem como objetivo organizar os exercícios de Programação Web e demonstrar a evolução prática em desenvolvimento front-end durante a disciplina.
